Day 17: Introduction to Forms


Web article comment (or contact) form, by default post to current page: name, email, url, text, submit button.

Wrap a form in a Form element, two attributes: action and method.

Method describes how form contents are transmitted to server side language.

The method="get" uses the url, or "post" for more security.

The action="#" (current page) is default or may specify, e.g. "contact.php".

Create text box using input element. Note self-closing differently. <input type="" />

For form action "", method "get", url appended with query string, each input separated by an ampersand.
