Changing the default text editor on macOS

In terminal, copy-and-paste the command from below that matches the text 
editor you want to use as your default. Then, press return to run it, and 
restart your machine.

# Sublime Text 3
defaults write com.apple.LaunchServices/com.apple.launchservices.secure 
LSHandlers -array-add 
'{LSHandlerContentType=public.plain-text;LSHandlerRoleAll=com.sublimetext.3;}'

# Sublime Text 4
defaults write com.apple.LaunchServices/com.apple.launchservices.secure 
LSHandlers -array-add 
'{LSHandlerContentType=public.plain-text;LSHandlerRoleAll=com.sublimetext.4;}'

# VS Code
defaults write com.apple.LaunchServices/com.apple.launchservices.secure 
LSHandlers -array-add 
'{LSHandlerContentType=public.plain-text;LSHandlerRoleAll=com.microsoft.VSCode;}'
